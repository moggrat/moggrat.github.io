Background

I’m Luke Scott a Computer Science graduate from NW England, and I build all sorts of programs, websites and video games.

To develop these I've used a variety of technologies and skills including: Java, C#, HTML, CSS, JS, Photoshop, Paintdotnet, WordPress, Symphony, Python, Git, Flexbox, Jekyll, Blender, and Unity. (Already a long list so I'll stop there).

Alongside developing games/projects I enjoy playing games, speedrunning, walking/cycling, and going to gigs/festivals.


My Public Projects

  • Rocsoc.org: This is the website for the Lancaster University Rock and Metal society.
  • Civilization VI Shuffle & Randomization: This is a web tool to help randomize settings in civilization VI games.
  • MOON MISS SHOT: This was my entry into the GitHub Game Off 2020 game jam. The theme was MOONSHOT. This theme alongside other inspirations resulted in me developing a gravity system and the game around this system.
  • Hazardman: This was my entry into the ANYTHING GOES JAM (game jam). I joined the jam late, but I was able to create a rough prototype in those two days.

Other Stuff I’ve worked on

  • Multiple websites built using WordPress and Symphony.
  • LNSA. This was a simple application meant to display the output of the command “ipconfig” in a visually pleasant manner.
  • Final Year Project. Using Unity and Socket.io I created a 6+ player multiplayer game. Players controller their “duck” using a webpage on their smartphones.
  • Second Year Group Project. Using an old Java API we created a 3 level shoot ‘em up space game. This game supported 2 players (on one keyboard).
  • College Project. I created a simple rogue-like prototype using monogame.
